# CoinSeek - a glam_račke app

CoinSeek is the entry application from the glam_račke team for the [DragonHack](https://dragonhack.si/) 2024 hackathon at [FRI](https://fri.uni-lj.si/sl) in Ljubljana, Slovenia.
